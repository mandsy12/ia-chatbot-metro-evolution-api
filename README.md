# Chatbot de IA - Projeto Integrador Metrô

Este repositório contém a primeira entrega da matéria de Inteligência Artificial.

## Objetivo

Criar um chatbot simples utilizando a Evolution API para comunicação via WhatsApp.

Nesta etapa, o chatbot ainda não possui integração direta com o sistema principal do Projeto Integrador. Ele funciona como um protótipo inicial de atendimento automatizado.

## Projeto Integrador

O projeto principal consiste em um sistema de autoatendimento para compra de bilhetes de metrô por meio de um totem.

## Tecnologias utilizadas

- Node.js
- Express
- Axios
- Dotenv
- Evolution API
- Webhook
- WhatsApp

## Funcionalidades

- Receber mensagens via WhatsApp
- Interpretar comandos simples
- Responder automaticamente
- Utilizar webhook para integração com a Evolution API
